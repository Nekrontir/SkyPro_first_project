# Проект банковского приложения

## Цель

Это ***учебный*** проект. Приложение разрабатывается для закрепления и практики
полученных ранее навыков во время занятий. 
Заполнение **Readme** будет происходить по мере новых итераций разработки. 

## Установка

* Скопировать репозиторий на ПК:
`git clone https://github.com/Nekrontir/SkyPro_first_project/tree/main ` 
* Для установки виртуального окружения и всех зависимостей в проекте используется Poetry:
`poetry install`

## Содержание проекта

На данный момент проект на стадии разработки и содержит модули:
1. generators.py - содержит функции-генераторы
2. masks.py - содержит функции предназначены для маскировки номера счёта или карты
3. processing.py - содержит функции для фильтрации по состоянию или дате
4. widget.py - содержит функции для обработки строк содержащих время и номер счёта или карты
5. decorators.py - содержит функции декораторы (логирование)


## Тесты

В репозиторий добавлены тесты. Используется фреймворк PyTest.
На данный момент тестами покрываются все функции.
Для запуска тестов с автоматическим формированием результатов необходимо воспользоваться командой:  
` pytest --cov=src --cov-report=html`
